# PSGIN
This is a PyTorch implementation of "PSGIN".

## Requirements
To run the code, you need to install `Python(>=3.9.12)` and `PyTorch(>=1.11.0)` at least.

## Data
The datasets can be downloaded from [GoogleDrive](https://drive.google.com/drive/folders/1qSESYIfs2UQdqdIXh22scVxBbcrxvzby?usp=sharing) and [Weather5K](https://github.com/taohan10200/WEATHER-5K?tab=readme-ov-file)
